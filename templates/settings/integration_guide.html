Landing Page Integration Guide

Connect your landing page (Porkbun, custom HTML, etc.) to automatically capture leads into the CRM.

2. API Endpoint

POST {{ request.scheme }}://{{ request.get_host }}/api/leads/

Required Header:

X-Api-Key: {% if api_key %}{{ api_key.key }}{% else %}your-api-key-here{% endif %}

3. Request Body (JSON)

{
    "first_name": "John",
    "last_name": "Doe",
    "email": "john@example.com",
    "phone": "555-123-4567",
    "source": "landing_page",
    "utm_source": "facebook_ads",
    "campaign_id": 1
}

Fields:

  • first_name — Contact's first name
  • last_name — Contact's last name
  • email — Contact's email address
  • phone — Contact's phone number
  • source — Lead source (default: "landing_page")
  • utm_source — UTM source for tracking (optional)
  • campaign_id — Auto-enroll in a drip campaign (optional)

4. HTML Form Snippet

Copy this form into your Porkbun landing page or any HTML page:

<form id="lead-form">
    <input type="text" name="first_name" placeholder="First Name" required>
    <input type="text" name="last_name" placeholder="Last Name" required>
    <input type="email" name="email" placeholder="Email" required>
    <input type="tel" name="phone" placeholder="Phone">
    <input type="hidden" name="source" value="landing_page">
    <button type="submit">Get Started</button>
</form>

<script>
document.getElementById('lead-form').addEventListener('submit', async function(e) {
    e.preventDefault();
    const formData = new FormData(this);
    const data = Object.fromEntries(formData.entries());

    try {
        const response = await fetch('{{ request.scheme }}://{{ request.get_host }}/api/leads/', {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json',
                'X-Api-Key': '{% if api_key %}{{ api_key.key }}{% else %}YOUR_API_KEY{% endif %}'
            },
            body: JSON.stringify(data)
        });

        if (response.ok) {
            alert('Thank you! We will be in touch soon.');
            this.reset();
        } else {
            alert('Something went wrong. Please try again.');
        }
    } catch (error) {
        alert('Something went wrong. Please try again.');
    }
});
</script>

6. Campaign Auto-Enrollment

Include a campaign_id in the request body to automatically enroll new leads into a drip campaign. The campaign must be active and belong to your team.

Find your campaign IDs in the Campaigns section. The ID is shown in the URL when you view a campaign (e.g., /campaigns/3/ means campaign_id is 3).

7. Test with cURL

Test the integration from your terminal:

curl -X POST {{ request.scheme }}://{{ request.get_host }}/api/leads/ \
  -H "Content-Type: application/json" \

    "email": "test@example.com",
    "phone": "555-000-0000",
    "source": "landing_page"
  }'

Expected response (201 Created):
